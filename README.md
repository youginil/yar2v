# yar2v

V2ray CLI client

## Warning!!!

The project is not allowed for illegal intention

## Features

-   Auto pulling servers from subscribed address
-   Auto checking server availablility

## Requirement

-   NodeJS
-   v2ray in `~/.yar2v/`

## Installation

```
pnpm install; npm run build; npm run dev
```
